# Export

Export your family tree data in standard formats for backup, sharing, or use in other genealogy software.

## GEDCOM Export

GEDCOM is the universal exchange format for genealogy data. Ancstra supports two versions:

### GEDCOM 5.5.1
The most widely supported version. Compatible with virtually all genealogy software including Ancestry, FamilySearch, Gramps, and Legacy Family Tree.

### GEDCOM 7.0
The modern standard with improved character encoding (UTF-8), better date handling, and structured source citations. Use this when sharing with software that supports the newer format.

## Export Options

When exporting, you can customize what is included:

- **All persons** or a filtered subset (by branch, generation, or surname)
- **Include/exclude living persons** — living persons can be redacted or omitted entirely
- **Include/exclude notes** — personal research notes can be stripped from the export
- **Include media references** — links to attached photos and documents

To export:

1. Go to **Settings > Export**
2. Choose your format (GEDCOM 5.5.1 or 7.0)
3. Configure include/exclude options
4. Click **Download**

## PDF Export

Generate a formatted PDF report of your family tree using Gotenberg (a document generation service):

- **Pedigree chart** — ancestor-focused view (4 or 5 generations)
- **Descendant report** — narrative-style descendant list
- **Family group sheets** — one page per family unit

PDF exports respect the living person filter — redacted persons show as "Living [Surname]" in the output.
