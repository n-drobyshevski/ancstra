# Persons

Persons are the core records in your family tree. Each person has a profile with biographical details, events, sources, and relationships.

## Add a Person

1. Click **Add Person** from the toolbar or right-click the canvas
2. Enter the person's name (given name and surname)
3. Optionally set gender and add a profile note
4. Click **Save** — the person appears on the tree

## Events

Events capture key life milestones. Supported event types include:

- **Birth** — date and place of birth
- **Death** — date, place, and cause
- **Marriage** — date, place, and spouse
- **Burial** — date and cemetery/place
- **Immigration / Emigration** — date, origin, and destination
- **Census** — year, location, and household details
- **Custom events** — any user-defined event type

Each event supports exact dates, date ranges, and approximate dates (e.g., "about 1842", "between 1850 and 1855").

## Sources and Citations

Attach sources to any fact or event to document your evidence:

1. Open the person's detail panel and navigate to the event
2. Click **Add Source** and select an existing source or create a new one
3. Add a citation detail (page number, entry number, etc.)

Sources follow a master source / citation model. One source record (e.g., "1850 U.S. Census") can have many citations across different persons.

## Relationships

Link persons together as parent-child or spouse pairs. Relationships are bidirectional — adding a parent to a child also adds the child to that parent. All relationships from external sources (AI research, record hints) go through a validation queue before being applied.
