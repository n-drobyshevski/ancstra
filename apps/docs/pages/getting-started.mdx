# Getting Started

## Create an Account

Sign up with your email address or use a social login provider. Once signed in, you land on your dashboard where you can manage all your family trees.

## Import a GEDCOM File

Already have genealogy data? Import it directly:

1. Click **New Tree** from the dashboard
2. Select **Import GEDCOM**
3. Choose your `.ged` file (GEDCOM 5.5.1 and 7.0 are supported)
4. Review the import summary — person count, family count, and any warnings
5. Click **Confirm Import**

Imported records are auto-trusted and marked as `confirmed` in the validation pipeline. You can review and edit them at any time.

## Start from Scratch

Prefer to build manually?

1. Click **New Tree** and select **Start Empty**
2. Name your tree (e.g., "Smith Family")
3. Add your first person — typically yourself or the earliest known ancestor
4. Add events (birth, death, marriage) and link family members as you go
